# Project Brief: PharmaNet MyShop

## Overview
Malaysian e-commerce platform for pharmaceutical products with:
- Consumer facing storefront
- Vendor management portal  
- Admin dashboard
- Malaysian-specific features (currency, address forms, etc.)

## Core Features
- User roles (admin, vendor, consumer)
- Product catalog with images
- Shopping cart & checkout
- Order management
- Malaysian localization
- Analytics & reporting

## Technical Stack
- Frontend: React, TypeScript, Tailwind CSS, Vite
- Backend: Supabase (PostgreSQL, Auth, Storage)
- Deployment: TBD

## Key Requirements
- Support Malaysian market needs
- Robust product management
- Secure authentication
- Scalable architecture
